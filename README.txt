Welcome to code-centered issue tracking!
--------------------------------------------

ciss is a simple command line tool for helping
to manage issues in a human-workable ISSUES.txt 
files.  

Use your favourite text editor for quickly
updating your projects issues - no tiresome
web interfaces required!

associate issues to file patterns in your project 
and assign tags for status/milestone/custom usage.

ciss is alpha software but has many automated tests.

have fun,
